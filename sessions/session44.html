1-3. 時間配分戦略

⏰ 時間配分の重要性

150分で90問を解答するには、効率的な時間配分が必須です。時間切れで見直しができない、または最後の問題まで到達できないという事態を避けましょう。

推奨時間配分

ブロック/活動 問題数 配分時間 1問あたり
Aブロック 30問 45分 約1.5分/問
Bブロック 30問 45分 約1.5分/問
Cブロック 30問 45分 約1.5分/問
見直し時間 - 15分 -
合計 90問 150分 約1.6分/問

💡 時間配分のコツ

  1. 分からない問題は一旦飛ばす
    • 1問に3分以上かけない
    • 悩む問題はマーク（フラグ）機能を使って印を付ける
    • 後で戻って考える時間を確保
  2. マーク機能を積極的に活用
    • CBT試験では、各問題に「マーク（旗のアイコン）」を付けられる
    • 自信がない問題、後で見直したい問題にマークを付ける
    • 見直し時はマークした問題から優先的に確認
  3. 見直し時間を必ず確保
    • 最低でも15分は見直し用に残す
    • マークした問題の再確認
    • 選択ミスや読み間違いの確認
    • 時間があれば全問題を再確認
  4. 各ブロックの時間を意識
    • Aブロック終了時点で45分経過を確認
    • Bブロック終了時点で90分経過を確認
    • Cブロック終了時点で135分経過が理想
    • 遅れている場合は、残りのペースを上げる
  5. 時計を定期的に確認
    • CBT試験では画面に残り時間が表示される
    • 10問ごとに時間を確認（10問で約16分ペース）
    • ペースが遅れていたら意識的に速める

⚠️ 時間配分での注意点

  • 完璧主義は禁物：1問に時間をかけすぎて、最後まで到達できないのは避ける
  • 早く解きすぎない：見直し時間が60分以上残るのは、急ぎすぎの可能性あり
  • 見直し時間ゼロは危険：ケアレスミスを見つけるチャンスを逃す
  • 残り30分で焦らない：焦ると誤答が増える。落ち着いて解答

🎯 模擬試験での時間配分練習

本講座の模擬試験では、実際の試験と同じ150分で解答してください。以下の手順で練習しましょう：

  1. タイマーをセット：150分（2時間30分）のタイマーを用意
  2. 解答開始：Aブロックから順番に解答（問1から問90まで）
  3. 中間チェック：45分・90分・135分経過時に進捗を確認
  4. マークの活用：自信がない問題はメモに問題番号を記録
  5. 見直し：最後の15分でマークした問題と全体を確認
  6. 時間記録：実際にかかった時間を記録して分析

第2章：模擬試験（90問）

【問69】Wi-Fiの暗号化方式

正解：2（WPA3）

解説：

Wi-Fiの暗号化方式の変遷：

  • WEP：古い規格、脆弱性あり、使用禁止
  • WPA：WEPの改良版、現在は非推奨
  • WPA2：長年の標準、まだ広く使用されている
  • WPA3：2018年登場、現在の推奨規格

WPA3の特徴：

  • より強力な暗号化
  • パスワード推測攻撃への耐性向上
  • 公衆Wi-Fiでのセキュリティ強化
  • 個別のデータ暗号化（OWE: Opportunistic Wireless Encryption）
【問72】バックアップの3-2-1ルール

正解：1（3つのコピー、2種類の媒体、1つはオフサイト）

解説：

バックアップの基本原則「3-2-1ルール」：

  • 3：3つのコピーを保持
    • 1つ：本体データ
    • 2つ：バックアップコピー
  • 2：2種類の異なる媒体に保存
    • 例：HDD + クラウド
    • 例：サーバー + 外付けHDD
  • 1：1つはオフサイト（別の場所）に保管
    • 災害・盗難対策
    • クラウドストレージの活用

学校での適用例：校務サーバー（本体）+ 校内バックアップサーバー + 教育委員会のクラウドストレージ

学習ポイント

著作権法第35条の理解

  • 適用範囲：
    • 授業の過程での使用に限定
    • 学校その他の教育機関（営利目的を除く）
    • 教員及び授業を受ける者
  • 利用できる範囲：
    • 複製：必要最小限の部数
    • 公衆送信：2020年4月改正で可能に
    • 必要と認められる限度
  • 補償金制度：
    • SARTRAS（授業目的公衆送信補償金等管理協会）
    • 小学校：120円/人/年
    • 中学校：180円/人/年
    • 高等学校：420円/人/年
  • 利用できないケース：
    • 著作権者の利益を不当に害する場合
    • 市販の教材等をそのまま代替する場合
    • 授業と関係のない目的での使用
個人情報保護法の理解

個人情報の定義

  • 特定の個人を識別できる情報：
    • 氏名、生年月日、住所
    • 顔写真、映像
    • 学籍番号（単独で個人を識別できる場合）
    • 他の情報と組み合わせて個人を識別できる情報
  • 個人情報の適正な取り扱い：
    • 利用目的の特定と通知
    • 適正な取得
    • 安全管理措置
    • 第三者提供の制限
  • 2023年4月改正のポイント：
    • 漏えい等の報告義務
    • 本人への通知義務
    • 個人関連情報の第三者提供規制
頻出キーワード
  • 授業目的公衆送信補償金制度：オンライン授業等での著作物利用に対する補償
  • SARTRAS：補償金を一括管理する指定管理団体
  • 肖像権：自分の顔や姿を無断で撮影・公表されない権利
  • クリエイティブ・コモンズ：著作物の利用条件を明示するライセンス
  • 引用：正当な範囲内、出典明示、主従関係が明確

分野2：技術的知識（Cブロック）

ネットワークの基礎知識

必須用語とその意味

  • IPアドレス：
    • ネットワーク上の機器を識別する番号
    • IPv4：192.168.1.1のような形式
    • IPv6：次世代規格、アドレス数が豊富
  • DNS（Domain Name System）：
    • ドメイン名（www.example.com）とIPアドレスを変換
    • インターネットの「電話帳」のような役割
  • LAN（Local Area Network）：
    • 校内ネットワーク
    • スター型配線が一般的（各端末がハブに接続）
  • Wi-Fi：
    • 無線LANの規格
    • 802.11ac（Wi-Fi 5）、802.11ax（Wi-Fi 6）
    • GIGAスクールでは無線LAN環境が標準
  • 帯域（バンド幅）：
    • データ通信速度の最大値
    • 学校では100Mbps〜1Gbps以上が望ましい
    • 1人1台端末を前提とした容量確保
セキュリティ対策の基礎

情報セキュリティの3要素（CIA）

  • C：Confidentiality（機密性）
    • 許可された者のみがアクセスできる
    • 対策：暗号化、アクセス制限、パスワード管理
  • I：Integrity（完全性）
    • 情報が改ざんされていない
    • 対策：デジタル署名、ハッシュ値チェック
  • A：Availability（可用性）
    • 必要な時に情報やシステムが利用できる
    • 対策：冗長化、バックアップ、障害対策

主なセキュリティ対策

  • ファイアウォール：不正アクセスを防ぐ「防火壁」
  • ウイルス対策ソフト：マルウェアの検出・駆除
  • 暗号化：
    • WPA3：Wi-Fiの暗号化（推奨）
    • SSL/TLS：Webサイトの通信暗号化（https://）
  • VPN（Virtual Private Network）：インターネット上の仮想専用線
  • アクセス制限：ユーザー権限の適切な設定
クラウドサービスの理解

クラウドの3つのサービスモデル

  • SaaS（Software as a Service）：
    • ソフトウェアをインターネット経由で提供
    • 例：Google Workspace、Microsoft 365
    • 学校での利用：校務支援システム、学習eポータル
  • PaaS（Platform as a Service）：
    • アプリケーション開発・実行環境を提供
    • 例：Google App Engine、Microsoft Azure
  • IaaS（Infrastructure as a Service）：
    • サーバー、ストレージなどのインフラを提供
    • 例：Amazon Web Services（AWS）、Google Cloud

クラウドのメリット・デメリット

  • メリット：
    • 場所を選ばないアクセス
    • 自動バックアップ
    • 複数端末での同期
    • 初期投資が少ない
  • デメリット・注意点：
    • インターネット接続が必須
    • データの保存場所の確認
    • 利用規約の確認
    • 個人情報保護への配慮

📚 学習方法

  • 用語集作成：専門用語とその意味を一覧にまとめる
  • 図解で理解：ネットワーク構成図、セキュリティの仕組みを図で理解
  • 実機で確認：自宅のWi-Fi設定、パソコンのIPアドレスを確認してみる
  • 類似用語の整理：WPA2とWPA3、SaaSとPaaSなど、混同しやすい用語を整理

💻 CBT（Computer Based Testing）とは

コンピュータで受験する試験形式です。紙の試験と異なり、画面上で問題を読み、マウスで選択肢をクリックして解答します。

CBT試験の特徴

項目 内容
コンピュータで受験 マウスとキーボードで操作。パソコンに不慣れでも操作は簡単
即時採点 試験終了後すぐに合否が分かる。スコアレポートが画面に表示
柔軟な受験日程 試験期間内で都合の良い日時を選択可能
全国のテストセンター 最寄りの会場で受験可能。予約制
試験中の時間表示 画面に残り時間が常に表示される
マーク機能 後で見直したい問題に印を付けられる

7-3. 操作のポイント

1. マーク機能の活用

  • マークボタン：自信がない問題、後で見直したい問題に旗のマーク（⚑）を付ける
  • マークの表示：ナビゲーションパネルで⚑マークが表示される
  • 見直し時：マークした問題だけを効率的に確認できる
  • 活用例：
    • 3分考えても分からない問題にマーク→一旦飛ばす
    • 2つまで絞り込んだが迷っている問題にマーク→後で再考
    • 見直し時にマーク済み問題から優先的に確認

2. 問題番号で直接移動

  • ナビゲーションパネル：画面下部に全問題番号が表示
  • クリックでジャンプ：番号をクリックすると任意の問題にジャンプ
  • 活用例：
    • 問50まで解答後、問1に戻って見直し
    • Aブロック（問1〜30）の見直しをまとめて行う
    • マーク済み問題に素早くアクセス

3. 選択肢の変更

  • 何度でも変更可能：一度選択した選択肢は何度でも変更できる
  • 変更方法：別の選択肢のラジオボタンをクリックするだけ
  • 注意点：
    • 変更する際は慎重に。最初の直感が正しいことも多い
    • 明確な理由がある場合のみ変更

4. 見直し時間の確保

  • 最低15分：見直し用に最低15分は残す
  • 見直し手順：
    1. マークした問題を優先的に確認（5〜10分）
    2. 全問題を一通り確認（5〜10分）
    3. 選択ミスがないかチェック
  • 時間管理：135分経過時点で90問目を終えるのが理想

8-2. 見直しの手順

📝 15分間の見直し手順

残り15分で効率的に見直しを行う手順です。

ステップ1：マークした問題の見直し（5〜7分）

  • ナビゲーションパネルで⚑マークの問題を確認
  • 問題文を再度読み直す
  • 選択肢を比較して、最も適切なものを選ぶ
  • 消去法で明らかに違うものから除外
  • それでも迷う場合は最初の直感を信じる

ステップ2：全問題の確認（5〜7分）

  • 問1から順番に、または各ブロックごとに確認
  • 問題文の読み間違いがないか
  • 「適切なもの」「不適切なもの」の確認
  • 選択ミス（意図した選択肢がマークされているか）
  • 未解答の問題がないか

ステップ3：最終チェック（1〜3分）

  • 全90問に解答しているか確認
  • 特に自信がない問題を最後にもう一度確認
  • 残り時間を確認
  • 落ち着いて「試験終了」ボタンをクリック

8-3. 見直し時の注意点

⚠️ 見直し時の注意点

  • 直感を信じる：
    • 最初の解答が正解のことが多い
    • 明確な理由がない限り変更しない
    • 「なんとなく違う気がする」だけでは変更しない
  • 変更は慎重に：
    • 変更する理由を明確に
    • 問題文を読み間違えていた→変更OK
    • 選択肢を押し間違えた→変更OK
    • 単なる不安→変更しない
  • 時間配分を守る：
    • 見直しに15分以上かけてもOK
    • ただし、見直し時間ゼロは避ける
    • 1問に5分かけるよりも、全問題をさっと確認
  • 焦らない：
    • 残り時間が少なくても焦らない
    • 焦ると誤答が増える
    • 深呼吸してリラックス

9-2. 試験中の心構え

1. 焦らない

  • 分からない問題は飛ばす：
    • 1問に3分以上かけない
    • マーク機能を活用
    • 後で戻って考える時間を確保
  • 時間が足りなくなっても：
    • パニックにならない
    • 残りの問題を冷静に解答
    • すべての問題に解答することを優先

2. 深呼吸でリラックス

  • 緊張したら：
    • 目を閉じて深呼吸（3回）
    • 肩の力を抜く
    • 「大丈夫、できる」と自分に言い聞かせる
  • 休憩タイミング：
    • 30問ごと（各ブロック終了時）
    • 10秒程度の深呼吸
    • 目を休める

3. 集中力維持

  • 150分間の集中力配分：
    • 前半（0〜45分）：Aブロック、やや緊張
    • 中盤（45〜90分）：Bブロック、集中力ピーク
    • 後半（90〜135分）：Cブロック、疲労が出る→深呼吸
    • 見直し（135〜150分）：最後の集中
  • 集中力が切れたら：
    • 深呼吸
    • 軽く首を回す
    • 水を飲む（許可されている場合）

4. 前向きな思考

  • ポジティブな自己暗示：
    • 「やるべきことはやった」
    • 「できる問題から確実に解答する」
    • 「分からない問題があっても大丈夫」
    • 「70%正解すれば合格できる」
  • ネガティブ思考の切り替え：
    • 「全然分からない」→「分かる問題もある」
    • 「時間が足りない」→「残り時間でできることをやる」
    • 「失敗した」→「次の問題に集中する」
