第9回：情報モラル・デジタルシチズンシップ・著作権

全10回中の第9回 | 学習時間目安：2時間

🎯 学習目標

  • 情報モラル教育の5つの柱を理解できる
  • デジタルシチズンシップの概念を説明できる
  • 教育現場での著作権（第35条）を理解できる
  • 児童生徒への指導支援の方法を知る

1. 情報モラル教育の5つの柱

📌 情報モラルとは

情報社会で適正な活動を行うための基となる考え方と態度。情報通信ネットワーク上の倫理、マナー、法令遵守、情報セキュリティ、健康面への配慮など。

1.1 5つの柱（文部科学省「情報モラル教育」より）

①情報社会の倫理

  • 他者への影響を考えた行動
  • ネット上でも人を傷つけない
  • 誹謗中傷、いじめの禁止
  • 相手の立場に立つ

②法の理解と遵守

  • 著作権の尊重
  • 肖像権の理解
  • 個人情報保護
  • 違法行為（不正アクセス、違法ダウンロード等）の認識

③安全への知恵

  • インターネット上の危険を回避
  • 不適切な情報に触れない
  • SNSでの出会い系被害防止
  • フィッシング詐欺への対応

④情報セキュリティ

  • パスワード管理
  • ウイルス対策
  • 個人情報の適切な取り扱い
  • SNSの公開範囲設定

⑤公共的なネットワーク社会の構築

  • 情報発信者としての責任
  • よりよい社会を作るためのICT活用
  • 誤情報・偽情報への対応
  • デジタルデバイド（情報格差）の理解

2. デジタルシチズンシップ

2.1 情報モラルとの違い

💡 2つのアプローチ

情報モラル（従来型）

  • 「〜してはいけない」という禁止・制限中心
  • 危険回避、トラブル防止
  • 受け身的

デジタルシチズンシップ（新しい考え方）

  • 「〜しよう」という前向き・創造的な活用
  • 社会参画、問題解決
  • 主体的

※ 両方のアプローチをバランスよく取り入れることが重要

2.2 前向きで創造的な活用

  • ICTで社会課題を解決
  • 自分の考えを発信
  • グローバルにつながる
  • クリエイティブな表現

2.3 社会参画の視点

  • デジタル社会の一員としての責任
  • より良い社会を作るためのICT活用
  • 多様性の尊重
  • 批判的思考（クリティカルシンキング）

3. 著作権の基礎

3.1 著作権法の基本

📌 著作権とは

著作物（思想や感情を創作的に表現したもの）を創作した人（著作者）に与えられる権利です。

  • 著作物の例：小説、音楽、絵画、写真、映画、プログラム、地図など
  • 保護期間：原則として著作者の死後70年（2018年改正により50年から延長）
  • 権利：複製権、公衆送信権、翻案権など

3.2 教育現場での著作権（第35条）

著作権法第35条により、教育現場では著作物を一定の条件下で利用できます。

💡 第35条で認められる利用

  • 授業の過程での使用 - 授業で必要な範囲内
  • 必要と認められる限度 - 全部コピーは原則NG
  • 無償 - 対価を受け取らない
  • 出所明示 - 引用元を明記

具体例

  • 教科書の一部をコピーして配布 ✅
  • 新聞記事をスキャンしてプレゼン資料に ✅
  • 音楽CDを授業で再生 ✅

3.3 SARTRAS（授業目的公衆送信補償金制度）

2020年度からスタートした制度で、オンライン授業等での著作物利用を円滑にします。

  • 対象：オンライン授業、メール配信、クラウド共有等
  • 補償金：教育機関が一括支払い（児童生徒1人あたり年額数百円）
  • できること
    • 授業資料をオンラインで配信
    • 教科書をPDF化して共有
    • 動画教材を配信
  • できないこと
    • 授業と関係のない利用
    • ドリル・問題集の大量コピー
    • 市販の教材の代替

3.4 クリエイティブ・コモンズ（CC）

著作者が「この条件なら自由に使ってOK」と意思表示する仕組みです。

📌 CCライセンスの種類

  • CC BY - 表示（作者名を表示すれば自由に使える）
  • CC BY-SA - 表示-継承（同じライセンスで公開なら OK）
  • CC BY-NC - 表示-非営利（商用利用は NG）
  • CC BY-ND - 表示-改変禁止（そのまま使うなら OK）

3.5 JAPAN SEARCH

国が運営する、著作権処理済みのデジタルアーカイブ統合ポータルです。

  • 全国の博物館、図書館、公文書館等のコンテンツを検索
  • 多くが二次利用可能
  • 授業での活用に最適

3.6 生成AIと著作権

生成AIが作ったコンテンツの著作権は複雑です。

⚠️ 注意点

  • AIが生成した画像・文章が著作物になるかは「人間の創作的寄与」の程度による（文化庁「AIと著作権に関する考え方について」2024年3月）
    • 単純なプロンプト入力のみ → 著作物性が認められない可能性が高い
    • 詳細な指示、試行錯誤、修正等 → 著作物性が認められる可能性
  • AIの学習データに他人の著作物が含まれている可能性
  • 生成物が既存の著作物に酷似する場合は著作権侵害のリスク
  • 安全な使い方：アイデア出し、下書き、要約等に留める

4. 児童生徒への指導支援

4.1 発達段階に応じた指導

💡 学年別の指導内容例

  • 小学校低学年
    • インターネットには良い情報も悪い情報もある
    • 困ったら大人に相談
    • 個人情報（名前、住所）は教えない
  • 小学校中学年
    • ネット上でも人を傷つけない言葉を使う
    • パスワードは他人に教えない
    • 写真を勝手に使わない（著作権・肖像権）
  • 小学校高学年
    • SNSの危険性（炎上、誹謗中傷）
    • 著作権の基本（引用のルール）
    • 情報の信頼性判断
  • 中学校
    • デジタルタトゥー（一度発信した情報は消えない）
    • フェイクニュースの見分け方
    • 依存症対策（適切な利用時間）

4.2 トラブル事例と対応

⚠️ よくあるトラブル

  1. SNSでのいじめ・誹謗中傷
    • 対応：すぐに教員・保護者に報告、スクリーンショット保存
  2. 個人情報の流出
    • 対応：該当箇所の削除依頼、保護者への連絡
  3. 著作権侵害
    • 対応：該当部分の修正・削除、著作権教育の再徹底
  4. 不適切な画像・動画の閲覧
    • 対応：フィルタリング強化、個別指導

4.3 保護者との連携

  • 家庭でのルール作りの支援
  • 保護者向け情報モラル研修
  • フィルタリング・ペアレンタルコントロール設定支援
  • トラブル時の迅速な情報共有

5. ICT支援員としての役割

💡 支援のポイント

  • 教材提供 - 情報モラル教材の紹介、提供
  • 授業支援 - 情報モラル授業での技術サポート
  • 研修企画 - 教職員向け著作権研修の企画
  • 相談対応 - 著作権に関する質問への回答
  • フィルタリング設定 - 適切なWebフィルタリング設定

📚 まとめ

  • 情報モラルは5つの柱（倫理、法、安全、セキュリティ、公共性）で構成
  • デジタルシチズンシップは前向き・創造的な活用を重視
  • 教育現場では著作権法第35条により一定の利用が認められる
  • SARTRASでオンライン授業での著作物利用が円滑に
  • ICT支援員は、情報モラル・著作権の正しい理解と実践をサポート
