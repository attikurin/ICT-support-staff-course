基礎編 第16回

授業支援の実践（中等教育）

所要時間: 90分 | レベル: 基礎

本日の目標

中学校・高等学校におけるICT活用の特性を理解し、教科担任制における効果的な授業支援の実践方法を習得する。

📘 学習目標

  • 教科担任制の特徴と支援のポイントを理解する
  • 中高生の自律性を活かした支援方法を習得する
  • 教科指導における専門的な支援ができるようになる
  • プレゼンテーション・探究学習の支援方法を理解する
  • オンライン・ハイブリッド授業の支援ができる
  • 進路指導・キャリア教育でのICT活用を支援できる

1. 中学校・高等学校の特性

1.1 教科担任制への対応

💡 小学校との違い

  • 複数の教員が異なる教科を担当
  • 各教員の専門性とICTスキルに差がある
  • 教科ごとに異なるICT活用方法が求められる

ICT支援員の役割

  • 各教科の特性に応じた支援
  • 教員間の情報共有の促進
  • 教科横断的なICT活用事例の提供
  • 複数教員への効率的な支援体制の構築

💡 授業時間の違い

  • 中学校：1コマ50分（一般的）
  • 高等学校：1コマ50分または45分（学校により異なる）
  • 小学校（45分）より長く、集中力や内容の深さが増す

具体的な対応例

教科 ICT活用例
数学 グラフ作成ソフト、シミュレーションツール
理科 実験データの記録・分析、観察動画の活用
英語 音声録音・再生、オンライン辞書、発音練習アプリ
社会 地図ソフト、統計データの可視化、資料検索
国語 文章作成、推敲支援、デジタル資料の活用

1.2 生徒の自律性を活かす

中等教育段階の特徴

  • 自己管理能力の向上
  • 主体的な学習姿勢の形成
  • 複雑な課題への取り組み能力
  • デジタルネイティブ世代としての感覚

自律性を促す支援

  • 生徒自身での問題解決を促す
  • 操作方法を「教える」より「調べ方を示す」
  • トラブルシューティングスキルの育成
  • ピアサポート体制の構築支援

🎯 段階的な自立支援

第1段階：基本操作の習得（中1前期）
├─ 丁寧な説明と実演
└─ 練習時間の確保
第2段階：応用操作の習得（中1後期〜中2）
├─ ヒントの提供
└─ 生徒同士の教え合い促進
第3段階：自律的活用（中3〜高校）
├─ 最小限の介入
├─ 高度な活用方法の提示
└─ 創造的な活用の支援

1.3 思春期への配慮

⚠️ 中高生特有の配慮事項

  • プライバシーへの配慮
    • 他の生徒に見られないよう配慮
    • 個別相談は目立たない場所で
  • 自尊心への配慮
    • 「分からない」と言いにくい年代
    • さりげない支援を心がける
  • 情報モラル
    • SNSトラブルのリスク
    • 著作権・肖像権の意識

2. 教科指導における支援

2.1 数学・理科での高度な活用

数学でのICT活用

  • 動的幾何ソフト（GeoGebra / ジオジブラ等）
    • 関数のグラフを動的に変化させる
    • 図形の性質を視覚的に理解
    • 支援のポイント：ソフトの基本操作のレクチャー
  • 表計算ソフト
    • データ分析、統計処理
    • 関数グラフの作成
    • 支援のポイント：関数式の入力方法、グラフ作成手順
  • プログラミング
    • アルゴリズムの理解
    • 数学的概念のシミュレーション
    • 支援のポイント：Python（高校「情報Ⅰ」等）、Scratch（中学校技術・家庭科等）の活用支援

理科でのICT活用

  • 実験データの記録
    • デジタルカメラ、動画撮影
    • センサーによる計測データの収集
    • 支援のポイント：データの保存形式、整理方法
  • データ分析・可視化
    • 表計算ソフトでのグラフ作成
    • 実験結果の比較・考察
    • 支援のポイント：適切なグラフ種別の選択
  • シミュレーション
    • 化学反応、物理現象のシミュレーション
    • 天体の動き、生物の進化
    • 支援のポイント：シミュレーションソフトの操作

2.2 英語でのコミュニケーション重視

活動 ICT活用 支援のポイント
リスニング デジタル教材、YouTube、TED Talks 字幕機能、速度調整の使い方
スピーキング 音声録音、発音評価アプリ 録音機能の使い方、フィードバック方法
ライティング 文章作成ソフト、オンライン辞書 スペルチェック、翻訳ツールの適切な使い方
国際交流 Web会議、メール、SNS 接続テスト、時差の確認、マナー指導

2.3 社会科での資料活用

  • 地理分野
    • Google Earth、地図アプリの活用
    • 統計データの可視化（人口動態、貿易データ等）
    • 支援のポイント：レイヤー機能、データのインポート方法
  • 歴史分野
    • デジタルアーカイブの活用
    • 年表作成ツール
    • 支援のポイント：信頼できる情報源の見極め方
  • 公民分野
    • ニュースサイト、官公庁サイトの活用
    • アンケート作成・分析
    • 支援のポイント：情報の比較・検証方法

2.4 国語での表現力育成

  • 文章作成
    • 文書作成ソフトでの推敲
    • 構成の可視化（アウトライン機能）
    • 支援のポイント：校正機能、コメント機能の活用
  • デジタルポートフォリオ
    • 作品の蓄積と振り返り
    • 成長の可視化
    • 支援のポイント：整理方法、公開範囲の設定

3. プレゼンテーション指導

3.1 効果的なスライド作成支援

💡 良いプレゼンテーションの要素

  • 分かりやすい構成（序論・本論・結論）
  • 視覚的に見やすいデザイン
  • 適切な情報量（1スライド1メッセージ）
  • 効果的な図表・画像の活用
  • 著作権への配慮

段階的な指導

段階 内容 支援のポイント
初級 基本操作、テンプレート利用 スライド追加、文字入力、画像挿入
中級 デザインの工夫、構成の改善 配色、フォント、レイアウトのアドバイス
上級 アニメーション、動画埋め込み 効果的な演出、過度な装飾の抑制

3.2 発表技術の支援

  • 発表練習の録画
    • 自分の発表を客観的に見る
    • 改善点の発見
    • 支援：録画機能の使い方、振り返りシートの活用
  • タイマー・リモコンの活用
    • 時間管理の練習
    • スムーズな操作
    • 支援：機器の使い方、トラブル対応
  • 相互評価
    • オンラインフォームでの評価
    • コメント機能の活用
    • 支援：評価項目の設定、集計方法

3.3 よくある課題と対策

課題 対策
文字が多すぎる 「1スライド1メッセージ」の原則を伝える
箇条書きを推奨
デザインが派手すぎる 内容が伝わることが最優先
シンプルなデザインの良い例を示す
著作権侵害 引用ルールの説明
フリー素材サイトの紹介
時間オーバー 練習の重要性を伝える
タイマー機能の活用

4. データ活用とグラフ作成

4.1 表計算ソフトの活用

基本的なスキル

  • データ入力、セルの書式設定
  • 基本的な関数（SUM / 合計、AVERAGE / 平均、COUNT / カウント等）
  • 並べ替え、フィルター
  • グラフ作成（棒グラフ、折れ線グラフ、円グラフ）

※Microsoft Excelの日本語版では「=合計()」「=平均()」等の日本語関数名も使用可能です。Google スプレッドシートやLibreOffice Calcでは英語関数名を使用します。

4.2 統計データの分析支援

📋 支援例：社会科「人口問題」の授業

活動内容：政府統計データを使って、人口動態を分析

支援内容：

  1. 統計サイトからのデータダウンロード方法
  2. 表計算ソフトへのデータインポート
  3. 必要なデータの抽出（フィルター機能）
  4. グラフの種類の選択（推移→折れ線、比較→棒グラフ）
  5. グラフの見やすさの改善（軸ラベル、凡例）

4.3 適切なグラフの選び方

目的 推奨グラフ 使用例
時系列の変化 折れ線グラフ 気温の変化、人口の推移
量の比較 棒グラフ 都道府県別生産量、アンケート結果
割合・構成比 円グラフ エネルギー構成、支出割合
相関関係 散布図 身長と体重、気温と売上

5. 探究学習・課題研究の支援

5.1 探究学習のプロセス

🎯 探究学習の流れ

  1. 課題の設定 - 問いを立てる
  2. 情報の収集 - 資料・データを集める
  3. 整理・分析 - 情報を整理し、考察する
  4. まとめ・表現 - 成果をまとめ、発表する

5.2 各段階でのICT支援

段階 ICT活用 支援内容
課題設定 マインドマップ、思考ツール アイデア発散ツールの使い方
情報収集 Web検索、デジタルアーカイブ 効果的な検索方法、情報の信頼性評価
整理・分析 表計算、データベース データ整理方法、分析手法
まとめ・表現 プレゼン、動画、Webサイト 効果的な表現方法、ツールの選択

5.3 情報リテラシーの指導

情報の信頼性評価

  • 情報源の確認（誰が、いつ発信したか）
  • 複数の情報源での裏付け
  • 一次情報と二次情報の区別
  • バイアスの認識

引用・参考文献の書き方

  • 著作権の尊重
  • 適切な引用方法
  • 参考文献リストの作成
  • 盗用（コピペ）の防止

6. オンライン・ハイブリッド授業の支援

6.1 Web会議システムの活用

基本的な機能と支援

  • 接続テスト - 授業前の動作確認
  • 音声・映像設定 - マイク、カメラの設定確認
  • 画面共有 - 資料提示の方法
  • ブレイクアウトルーム - グループ分けの設定
  • チャット機能 - 質問・コメントの受付
  • 録画機能 - 授業の記録と共有

6.2 ハイブリッド授業の支援

💡 ハイブリッド授業とは

教室にいる生徒とオンラインで参加する生徒が同時に授業を受ける形態

技術的な準備

  • カメラの配置（黒板と教員が映るように）
  • マイク・スピーカーの設定（全員に声が届くように）
  • 資料の共有方法（画面共有、デジタル黒板）
  • 通信の安定性確認

授業中の支援

  • オンライン参加者の様子モニター
  • チャットでの質問対応
  • 接続トラブル時の迅速な対応
  • 教員へのタイミングの良い助言

6.3 オンライン授業特有の配慮

  • 生徒の集中力維持
    • 適度な休憩時間の設定（30〜40分ごとに5〜10分程度の休憩）
    • 双方向性の確保（質問、発言機会）
    • チャットやアンケートの活用
  • 孤立感の軽減
    • 小グループでの交流
    • 一人ひとりに声をかける
    • チームワーク活動の工夫

※オンライン授業では、対面授業よりも疲労が蓄積しやすいため、こまめな休憩と活動の切り替えが重要です。

7. 進路指導・キャリア教育

7.1 進路情報の収集支援

  • 大学・専門学校の情報検索
    • オープンキャンパス情報
    • 入試情報、過去問題
    • 支援：効率的な検索方法、情報の整理
  • 職業情報の探索
    • 職業紹介サイト、動画
    • 求人情報サイト
    • 支援：信頼できる情報源の紹介

7.2 eポートフォリオの活用

🎯 eポートフォリオとは

学習の成果や活動記録をデジタルで蓄積し、振り返りや進路選択に活用するもの

※2021年に大学入試共通のJAPAN e-Portfolioはサービスを終了しましたが、各学校・自治体が独自のポートフォリオシステムを導入し、学習の記録と振り返りに活用しています。

記録する内容

  • 学習の記録（レポート、作品、テスト結果）
  • 課外活動（部活動、ボランティア、資格取得）
  • 振り返り（気づき、成長の実感）
  • 目標設定と達成状況

ICT支援員の役割

  • ポートフォリオシステムの使い方指導
  • 効果的な記録方法のアドバイス
  • データの整理・分類支援
  • 振り返りの促進

7.3 オンライン進路相談

  • Web会議システムでの個別相談
  • 大学・企業とのオンライン交流
  • バーチャルオープンキャンパス
  • 支援：接続サポート、プライバシー保護

演習1：教科別支援計画

以下の授業に対して、事前準備・授業中・授業後の支援内容を計画してください。

授業内容：中学2年 理科「電流と磁界」
実験データをタブレットで記録し、グラフ化して考察する

演習2：プレゼンテーション支援

高校1年生が総合的な探究の時間で、地域課題についてプレゼンテーションを行います。以下の生徒にどのようなアドバイスをしますか？

  • A生徒：スライドに文字が多すぎて読みにくい
  • B生徒：インターネットから画像をコピーして使用している
  • C生徒：アニメーション効果を多用しすぎて、内容が入ってこない

演習3：ハイブリッド授業のトラブル対応

ハイブリッド授業中に以下のトラブルが発生しました。どのように対応しますか？

  1. オンライン参加の3名の生徒に教員の声が聞こえていない
  2. 画面共有中に、教員のPCがフリーズした
  3. ブレイクアウトルームで1グループだけ音声が途切れている

📌 まとめ

  • 中等教育は教科担任制であり、各教科の特性に応じた支援が必要
  • 中高生の自律性を尊重し、自分で解決する力を育てる支援を心がける
  • 各教科での高度なICT活用（数学・理科のシミュレーション、英語の発音練習等）を支援
  • プレゼンテーション能力の育成は重要であり、段階的な指導が効果的
  • データ活用・グラフ作成のスキルは、探究学習に不可欠
  • オンライン・ハイブリッド授業では、技術面と教育面の両方の支援が必要
  • 進路指導・キャリア教育でのICT活用も、重要な支援領域
