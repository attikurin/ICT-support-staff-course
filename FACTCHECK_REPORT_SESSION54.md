## ⚠️ 要修正事項

### 【修正1】Intel N100のベースクロック周波数

**箇所**: 第2章 2.6 CPU(中央処理装置)の実装

**現在の記述**:
```
Intel N100
ベース/ブースト: 0.8GHz / 3.4GHz
```

**問題点**:
Intel公式仕様書（Ark）によると、Intel N100のベースクロックは **1.0GHz** です。

**修正案**:
```
Intel N100
ベース/ブースト: 1.0GHz / 3.4GHz
```

**根拠**: Intel Ark - Intel® Processor N100
https://ark.intel.com/content/www/us/en/ark/products/231803/intel-processor-n100-6m-cache-up-to-3-40-ghz.html

### 【修正2】DDR4-2400の登場年

**箇所**: 第2章 2.7 メモリ(RAM)の実装

**現在の記述**:
```
DDR4-2400
登場年: 2016
```

**問題点**:
DDR4規格自体は2014年にJEDECで標準化され、DDR4-2400は2014〜2015年頃から製品化されました。2016年は普及期です。

**修正案**:
```
DDR4-2400
登場年: 2014
```

**根拠**: JEDEC Standard JESD79-4 (2014年9月発表)

### 【修正4】LPDDR4Xの転送速度

**箇所**: 第2章 2.7 メモリ(RAM)の実装

**現在の記述**:
```
LPDDR4X
転送速度: 17GB/s
```

**問題点**:
LPDDR4Xの転送速度は動作周波数によって異なります。一般的なLPDDR4X-4266の場合、**デュアルチャネル**で約34GB/s（シングルチャネルで約17GB/s）です。表記が不明確です。

**修正案1（推奨）**:
```
LPDDR4X
転送速度: 17〜34GB/s
```

**修正案2**:
```
LPDDR4X
転送速度: 17GB/s（シングルチャネル）
```

**根拠**: JEDEC LPDDR4X仕様書
- LPDDR4X-4266: 4266 MT/s × 16bit ÷ 8 = 8.5GB/s/チャネル
- デュアルチャネル: 8.5GB/s × 2 = 17GB/s（x32構成）
- クアッドチャネル: 8.5GB/s × 4 = 34GB/s（x64構成）

### 【修正5】NVMe SSD PCIe 4.0のTBW値

**箇所**: 第2章 2.8 ストレージの実装

**現在の記述**:
```
NVMe SSD(PCIe 4.0)
寿命: 1500TBW
```

**問題点**:
TBW（Total Bytes Written）は容量によって大きく異なります。1500TBWは2TBモデルでも高めの値です。

**修正案**:
```
NVMe SSD(PCIe 4.0)
寿命: 600〜1200TBW（容量により異なる）
```

**具体例**:
- Samsung 990 PRO 1TB: 600TBW
- Samsung 990 PRO 2TB: 1200TBW
- Samsung 990 PRO 4TB: 2400TBW

**根拠**: Samsung 990 PRO公式仕様書

---

### 【修正6】eMMCの読込速度

**箇所**: 第2章 2.8 ストレージの実装

**現在の記述**:


**根拠**: JEDEC eMMC 5.1仕様書、実製品ベンチマーク
