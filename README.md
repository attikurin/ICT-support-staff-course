# ICT支援員・企画員のための教育講座 Webサイト

## 📘 概要

日本の教育現場の実情に即した、ICT支援員・企画員向けの包括的な教育講座Webサイトです。全90回のカリキュラム（基本講座62回＋補完講座28回）を提供しており、基礎学習から資格取得、高度専門スキルまで一貫してサポートします。

## 🎯 対象者

- これからICT支援員を目指す方
- 現役のICT支援員（経験1年未満〜3年以上）
- 教育ICT企画員・推進リーダーを目指す方
- 教育委員会のICT担当者

## 🚀 使い方

1. **ブラウザで開く**
   ```
   index.htmlをブラウザで開いてください
   ```

2. **学習の進め方**
   - トップページから自分のレベルに合った編を選択
   - 各セッションは順番に学習することを推奨
   - 演習課題に取り組み、実践力を養う

3. **参考資料の活用**
   - resources.htmlから最新の公式情報にアクセス
   - 文部科学省、デジタル庁等の一次情報を確認

## ✨ 特徴

- **包括的なカリキュラム** - 全90回（基本62回+補完28回）で基礎から発展、高度専門スキルまで体系的に学習
- **実践重視** - 演習課題とグループワークで実践力を養成
- **最新動向対応** - 2024-2025年の最新情報を反映（生成AI、NEXT GIGA等）
- **試験対策完備** - ICT支援員能力認定試験・教育情報化コーディネータ検定試験（3級・2級）に完全対応
- **確認テスト** - 各編に確認テスト（80%以上で合格）
- **修了証発行** - 合格者には修了証を発行
- **レスポンシブデザイン** - PC、タブレット、スマートフォン対応
- **印刷対応** - 各ページを印刷して資料として活用可能

## 📋 セッションの構成

各セッションは以下の要素で構成されています：

1. **学習目標** - そのセッションで達成すべき目標
2. **本文コンテンツ** - 詳細な解説
3. **情報ボックス** - 重要ポイントの強調
4. **演習課題** - 理解を深めるための実践課題
5. **まとめ** - キーポイントの再確認
6. **参考資料** - さらに学ぶためのリソース
