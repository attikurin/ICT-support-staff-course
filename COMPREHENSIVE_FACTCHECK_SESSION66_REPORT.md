### 4-2. 差分の差分法（DID: Difference-in-Differences）

#### ✅ 理論的正確性（100/100）

**計算式の検証**:
```
効果 = (介入群の事後 - 介入群の事前) - (統制群の事後 - 統制群の事前)
```
✅ **判定**: 完全に正確。DIDの基本式。

**具体例の検証**:
```
介入群（A市）: 65点 → 75点 (+10点)
統制群（B市）: 64点 → 69点 (+5点)
DID推定値: +10 - (+5) = +5点
```
✅ **判定**: 計算が正確。

**前提条件の検証**:
| 前提条件 | 説明 | 検証結果 |
|---------|------|---------|
| **平行トレンド仮定** | 介入がなければ両群のトレンドは平行 | ✅ 正確 |
| **外生性** | 介入のタイミングが外生的 | ✅ 正確 |
| **SUTVA** | 一方の群が他方に影響を与えない | ✅ 正確（Stable Unit Treatment Value Assumption） |

**判定**: 完全合格。DIDの前提条件が明確。
