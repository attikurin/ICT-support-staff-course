### 1. 新規ページの作成

#### **contact.html** - お問い合わせページ

**ファイルサイズ**: 8,623 バイト  
**作成日時**: 2025-12-31 23:50

**主要機能**:
- Formspreeを利用したお問い合わせフォーム
- よくある質問（FAQ）の表示
- 直接メールでの問い合わせ情報

**フォーム項目**:
| 項目 | 入力タイプ | 必須 | 説明 |
|------|-----------|------|------|
| お名前 | text | ✓ | 問い合わせ者の氏名 |
| メールアドレス | email | ✓ | 返信先メールアドレス |
| 件名 | text | ✓ | 問い合わせの件名 |
| お問い合わせ種別 | select | - | カテゴリ選択（講座内容、受講方法、資格試験、補完カリキュラム、その他） |
| お問い合わせ内容 | textarea | ✓ | 詳細な問い合わせ内容 |

**Formspree設定**:
```html
<form action="https://formspree.io/f/mqeakdep" method="POST" class="contact-form">
    <input type="text" id="name" name="name" required>
    <input type="email" id="email" name="_replyto" required>
    <input type="text" id="subject" name="_subject" required>
    <select id="category" name="category">...</select>
    <textarea id="message" name="message" required></textarea>
</form>
```

**特別なフィールド名**:
- `_replyto`: 返信先メールアドレス（Formspreeの特別フィールド）
- `_subject`: メールの件名（Formspreeの特別フィールド）

## 🔧 技術詳細

### Formspreeの特徴

1. **設定不要のバックエンド**
   - サーバーサイドのプログラミング不要
   - HTMLフォームのみで実装可能

2. **スパム対策**
   - Formspree側で基本的なスパムフィルタリング
   - reCAPTCHA統合も可能（オプション）

3. **通知機能**
   - フォーム送信時に指定メールアドレスへ通知
   - 送信者への自動返信も設定可能

4. **無料プラン**
   - 月50件まで無料
   - 十分な容量（小規模サイト向け）

### セキュリティ考慮事項

1. **HTTPS通信**
   - Formspreeは自動的にHTTPSで通信

2. **個人情報保護**
   - プライバシーポリシーをフォーム下部に明記
   - 「お問い合わせへの回答のみに使用」と明示

3. **バリデーション**
   - HTML5のrequired属性による入力チェック
   - メールアドレスはtype="email"で形式チェック
